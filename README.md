# English Lesson - おしゃべり Shadow

英語シャドーイング練習アプリ。お手本音声を聴いて、声に出して真似する（シャドーイング）ことで英語の発音・リズム・イントネーションを鍛えます。

## デモ

https://github.com/tsubasagit/English-lesson/releases/download/v1.0.0/demo.mp4

<img src="docs/screenshot.png" width="320" alt="ホーム画面">

## 主な機能

- **シャドーイング練習** - 聴く → 録音する → 聞き比べる の3ステップ
- **TTS読み上げ** - お手本音声がなくても端末のText-to-Speechでスクリプトを再生
- **発音スコア** - 録音した音声をテキスト化し、お手本スクリプトとの一致率を表示
- **進捗トラッキング** - 練習回数・連続日数・平均スコア・週間カレンダー
- **シーン別レッスン** - 日常会話 / ビジネス / 旅行 / イディオム etc.

## 技術スタック

- **React Native** (Expo SDK 55)
- **TypeScript**
- **Expo Router** - ファイルベースルーティング
- **Zustand** - 状態管理
- **expo-av** - 音声録音・再生
- **expo-speech** - Text-to-Speech
- **Firebase** - 認証・データベース（準備中）

## セットアップ

```bash
# 依存関係をインストール
npm install

# 開発サーバーを起動
npx expo start
```

Expo Go アプリで QR コードを読み取るか、`w` キーでブラウザプレビューを起動できます。
